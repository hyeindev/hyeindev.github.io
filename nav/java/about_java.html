자바란? 프로그래밍 언어
jdk = java development kit
jre = java runtime environment
jvm = java virtual machine
jdk를 설치하면 호환되는 jre이 포함되어있고, jre는 자바클래스라이브러리(jcl),jvm을 포함하고있음


프로그램 실행 과정
프로그래밍을 하면 java 파일을 컴파일러가 기계어(바이트코드)로 변환하고, 이것을 jvm이 읽고 실행한다.

컴파일러(javac.exe)는 jdk에 포함되어있음

가상머신은 서로 다른 실행환경에서도 프로그램이 실행되게끔 해준다
마이크로소프트웨어의 .NET도 CLR(common language runtime)이라는 소프트웨어 가상기계에서 실행됨

jdk 설치 후 컴파일러와 jvm이 동작하는 과정을 직접 해보기
메모장에 아래 코드를 입력 후 (클래스명).java 파일로 저장해보자 
public class Test {
    public static void main(String[] args) {
        System.out.println("Hello world");
    }
}

cmd를 켜고 해당 java파일 경로에서 다음 명령어를 실행 
javac Test.java

컴파일러가 작동하여 .java 파일을 바이트코드로 작성된 .class 파일로 변환한것이 확인됨
이제 jvm을 동작시켜보자
java Test

프로그래밍한 대로 실행되는 것을 확인할 수 있다!
