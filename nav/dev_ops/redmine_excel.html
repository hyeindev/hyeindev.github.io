레드마인에 다이어그램 작성

05 Dec 2022 - use Plugin

  • git-bash 열어서 Bitnami\redmine-5.0.0-1_\apps\redmine\htdocs\plugins 경로 접근
  • 레드마인 플러그인 깃 저장소에서 클론 git clone -b main https://github.com/conveniencable/redmine_excel_connector https://github.com/mikitex70/redmine_drawio 에서 소스 클론



  • 루비 cmd 켜서 레드마인 저장소 이동 후 bundle exec rake redmine:plugins:migrate RAILS_ENV=production 명령어 실행

  • 레드마인 재시작

  • 엑셀 커넥터 다운로드
    reference : https://github.com/conveniencable/RedmineExcelConnectorRelease/releases/tag/V1.0.0.6 에서 설치파일 다운로드
    설치 방법 참고자료 : https://conveniencable.com/documents/excel/excel#install




  • 로컬에서 엑셀 실행 후 아래와 같은 팝업이 뜬다면 설치를 진행한다.




  • 엑셀 새 문서를 열면 생성 된 redmine 탭 클릭 후 open > 레드마인 URL 을 설정해준다.
