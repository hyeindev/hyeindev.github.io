29 Nov 2022 - 깃, 젠킨스 연동

  • Jenkins관리 > 시스템 설정





  • 페이지에서 Publish over SSH 검색 후 SSH Servers 설정
    • 따로 SSH 설치하고 퍼블릭 키를 발급 할 필요 없이 Window 서버 원격 계정을 사용하면 된다.
    Name 서버 별명 (아무거나 입력해도 된다.)
    Username 원격 서버의 접속 ID
    Remote Directory 원격 서버에 접속하여 작업하게 될 기준 디렉토리
    Passphrase / Password 원격 서버의 접속 Password



    원격 계정은 윈도우에서 제공하는 '원격 데스크톱 연결' 로 테스트 가능하다.




  • 여기까지 설정 끝났으면 우측 하단에 Test Configuration 버튼 클릭 > Success 문구가 뜨면 SSH 접속 성공이다.
